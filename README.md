# Restaurant Landing Page - Vue 3 + Vite + Tailwind CSS

Ce projet est une page d'accueil moderne, performante et entièrement responsive pour un restaurant, construite avec l'écosystème Vue 3.

## Technologies

- **Vue 3 (Composition API)** : Pour une interface réactive et modulaire.
- **Vite** : Pour un environnement de développement ultra-rapide.
- **Tailwind CSS** : Pour un stylage utilitaire et une maintenance simplifiée.
- **Lucide Vue Next** : Pour une iconographie élégante.

## Fonctionnalités

- **Design Mobile-First** : Navigation adaptée à tous les écrans (Header avec menu burger/liste pour mobile).
- **Identité Visuelle Forte** : Utilisation d'une palette de couleurs personnalisée (`brand-orange`, `brand-dark`).
- **Sections Clés** :
    - Navigation collante (Sticky Header).
    - Grille de plats spéciaux (en cours).
    - Section "Our Happy Customers" avec témoignages dynamiques.
    - Bannière de Newsletter.

## Identité Visuelle

Le thème est configuré dans `tailwind.config.js` avec les couleurs suivantes :
- **Orange** : `#ea6d27` (Accents et boutons)
- **Dark** : `#101a24` (Titres et navigation)
- **Gray** : `#5c5c5c` (Textes secondaires)

## Installation et Utilisation

1. **Installation des dépendances** :
   ```bash
   npm install
   ```

2. **Lancement du serveur de développement** :
   ```bash
   npm run dev
   ```

3. **Build pour la production** :
   ```bash
   npm run build
   ```
